# Local SEO Checklist
## Aim High Electrical Services

---

## Month 1: Foundation

### Citations
- [ ] Submit to Tier 1 directories (8 directories)
- [ ] Submit to Tier 2 industry directories (6 directories)
- [ ] Submit to Tier 3 local directories (5 directories)
- [ ] Verify NAP consistency across all submissions
- [ ] Track submission dates and verification status

### On-Page SEO
- [ ] Optimize title tag (under 60 chars with location)
- [ ] Optimize meta description (under 155 chars with CTA)
- [ ] Verify H1 includes primary keyword
- [ ] Check heading hierarchy (H1 > H2 > H3)
- [ ] Ensure NAP is in footer
- [ ] Verify Schema markup is rendering correctly
- [ ] Check page speed (target <3 seconds)
- [ ] Verify mobile-friendly

### Technical Setup
- [ ] Submit sitemap to Google Search Console
- [ ] Submit to Bing Webmaster Tools
- [ ] Verify robots.txt allows crawling
- [ ] Check for crawl errors in Search Console
- [ ] Ensure canonical URL is set

---

## Month 2-3: Growth

### Content
- [ ] Publish 2 blog posts per month (local + service focused)
- [ ] Create individual service page content (for future site expansion)
- [ ] Add location-specific content to landing page
- [ ] Optimize existing content based on Search Console data

### Citations
- [ ] Submit to Tier 4 general directories (5 directories)
- [ ] Check all Tier 1 citations for accuracy
- [ ] Fix any NAP inconsistencies found
- [ ] Monitor for duplicate listings

### Link Building
- [ ] Join Aurora Chamber of Commerce
- [ ] Explore veteran business directories
- [ ] Reach out to 3 potential referral partners
- [ ] Look for local sponsorship opportunities

## Ongoing (Monthly)

- [ ] Review Google Search Console for new keyword opportunities
- [ ] Check for new citation errors
- [ ] Publish 2 blog posts
- [ ] Monitor competitor rankings
- [ ] Review and respond to all new reviews
- [ ] Update Google Business Profile with fresh content
