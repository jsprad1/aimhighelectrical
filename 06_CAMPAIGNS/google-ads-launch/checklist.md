# Google Ads Launch Checklist
## Aim High Electrical Services

---

## Pre-Launch

### Strategy
- [ ] Campaign brief reviewed and approved by client (Glen)
- [ ] Budget confirmed ($500-750/month Month 1)
- [ ] Target audience defined (Aurora homeowners 30-65)
- [ ] Success metrics established (CPL <$50, 5+ leads/month)
- [ ] Competitor ad research completed

### Account Setup
- [ ] Google Ads account created
- [ ] Billing method configured
- [ ] Account linked to Google Analytics 4
- [ ] Account linked to Google Business Profile
- [ ] Time zone set to Mountain Time

### Tracking Setup
- [ ] Google Tag Manager installed on landing page
- [ ] Google Ads conversion tracking pixel placed
- [ ] Phone call click tracking configured
- [ ] Contact form submission tracking configured
- [ ] Call tracking service activated (CallRail recommended)
- [ ] Google Analytics 4 events configured
- [ ] Conversion actions verified in testing

### Campaign Build
- [ ] 4 campaigns created (General, Panel, EV, Emergency)
- [ ] Ad groups structured with themed keywords
- [ ] 40+ keywords added across campaigns
- [ ] Negative keyword list applied (commercial, jobs, DIY, etc.)
- [ ] Match types set (phrase match + exact match primary)
- [ ] Geographic targeting set (Aurora + 15-mile radius)
- [ ] Ad scheduling configured (7am-7pm Mon-Sat)
- [ ] Device bid adjustments set (mobile +10%)
- [ ] Daily budgets allocated per campaign

### Ad Copy
- [ ] 2-3 responsive search ads per ad group
- [ ] Headlines include: service, location, USP, CTA
- [ ] Descriptions include: benefits, credentials, phone number
- [ ] Display URLs customized per campaign
- [ ] Ad copy reviewed for policy compliance

### Ad Extensions
- [ ] Sitelink extensions added (6 links)
- [ ] Callout extensions added (8 callouts)
- [ ] Call extension configured (720-326-8492)
- [ ] Location extension linked to GBP
- [ ] Structured snippet extensions added (Services)
- [ ] Business logo uploaded

### Quality Assurance
- [ ] Landing page loads fast (<3 seconds)
- [ ] Landing page mobile-friendly
- [ ] Contact form functional (Formspree configured)
- [ ] Phone number clickable on mobile
- [ ] All ad URLs resolve correctly
- [ ] Tracking pixels firing correctly (use GTM preview)
- [ ] No policy violations in ad copy

---

## Launch Day

- [ ] All campaigns set to "Enabled"
- [ ] Confirm ads approved (no disapprovals)
- [ ] Verify tracking is firing (make test click)
- [ ] Check ad positions (search for target keywords)
- [ ] Set up Google Ads mobile app for monitoring
- [ ] Notify Glen that campaigns are live

## Post-Launch (24-48 Hours)

- [ ] Check all ads for approval status
- [ ] Review initial impression and click data
- [ ] Verify conversions are tracking properly
- [ ] Identify any obviously irrelevant search terms
- [ ] Add new negative keywords as needed
- [ ] Check that daily budgets aren't exhausting early
- [ ] Confirm geographic targeting is working correctly
